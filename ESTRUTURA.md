# Estrutura de Arquivos gerados

Após seguir o tutorial sua estrutura de arquivos deve ficar da seguinte maneira:

```
react-error-screen
├── node_modules
├── dist
├── .babelrc
├── package.json
├── webpack.config.js
├── yarn.lock
├── public
│   ├── index.html
│   └── index.js
└── src
    ├── index.js
    └── ErrorComponent.js
```
