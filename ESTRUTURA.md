# Estrutura de Arquivos gerados

Após seguir o tutorial sua estrutura de arquivos deve ficar da seguinte maneira:

```
minha-biblioteca
├── node_modules
├── .babelrc
├── package.json
├── webpack.config.js
├── yarn.lock
├── public
│   ├── index.html
│   └── index.js
└── src
    ├── index.js
    └── MyLibrary.js
```
